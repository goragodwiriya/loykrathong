# Loy Krathong — Static Website

คำอธิบายสั้น ๆ

โปรเจ็คนี้เป็นเว็บหน้าเดียว (static site) ที่ประกอบด้วยไฟล์ HTML, JavaScript และ CSS ซึ่งใช้สำหรับแสดงเนื้อหาเกี่ยวกับงานลอยกระทง (หรือเป็นตัวอย่างหน้าเว็บทั่วไป)

ไฟล์สำคัญในโปรเจ็ค

- `index.html` — หน้า HTML หลักของเว็บ
- `script.js` — สคริปต์ JavaScript สำหรับตรรกะฝั่งลูกค้า
- `styles.css` — สไตล์ (CSS) สำหรับหน้าเว็บ

ฟีเจอร์และพฤติกรรมที่คาดหวัง

- โหลดหน้า `index.html` แล้วแสดงเนื้อหาอย่างเรียบง่าย
- `script.js` อาจมีการโต้ตอบหรือการจัดการ DOM
- `styles.css` จัดการการแสดงผลของหน้า ให้ดูสวยงามบนหน้าจอทั่วไป

วิธีใช้งาน (ทดสอบอย่างรวดเร็ว)

1) เปิดไฟล์ `index.html` โดยตรงในเบราว์เซอร์

- บนระบบเดสก์ท็อป ให้ดับเบิลคลิกที่ `index.html` หรือใช้คำสั่ง `xdg-open` (Linux)

2) รันเว็บเซิร์ฟเวอร์แบบง่าย (แนะนำเมื่อใช้ฟีเจอร์ AJAX หรือเส้นทาง relative)

```bash
# ถ้าติดตั้ง Python 3
python3 -m http.server 8000
# แล้วเปิด http://localhost:8000/ ในเบราว์เซอร์

# หรือถ้าใช้ Node.js (ติดตั้ง http-server)
npm install -g http-server
http-server -p 8000
```

การพัฒนา

- แก้ไข `index.html`, `script.js`, และ `styles.css` ตามต้องการ
- รีโหลดหน้าในเบราว์เซอร์เพื่อตรวจการเปลี่ยนแปลง
